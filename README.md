# Basecamp Coffee — Coffee Personality Quiz

A web app built as part of the **Claude Code for Everyone** course by Carl Vellotti. This project is the hands-on capstone of Module 2: Vibe Coding.

## What This App Does

The Coffee Personality Quiz helps Basecamp Coffee loyalty members discover their coffee identity. Answer a few questions and get matched to a personality type — like "Espresso Explorer" or "Cozy Classic" — along with personalized drink recommendations.

It was built to solve a real problem: the Basecamp Rewards loyalty program had 12,400 sign-ups but only 17% engagement, because it had no personality. The quiz gives members an identity to connect with, turning a forgettable points program into something people actually talk about.
